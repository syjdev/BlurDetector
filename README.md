# Intro

__BlurDiscriminator__ is a project that based Deep-Learning for detection blurry image.<br />
BlurDiscriminator classify clear pixels & blurry pixels of image, as below.

<div align = "center">
<img src="https://user-images.githubusercontent.com/17686601/140477196-da561240-2644-4b1a-a2ff-f62e48b94452.png" width="80%"/>
</div>

<br />

Currently, You can use BlurDiscriminator for only [iOS Platform](Demo/iOS). Ones For Other Platforms, will be released.

# Usage

- [iOS](Demo/iOS)

# Dataset

For Learning Model, I used [Blur Detection Dataset](http://www.cse.cuhk.edu.hk/~leojia/projects/dblurdetect/dataset.html). <br />
